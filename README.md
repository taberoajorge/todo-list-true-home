# Todo List Practice

Aplicación de lista de tareas con React.

## Tecnologías

- React
- JavaScript

## Instalación

```bash
npm install
npm start
```

## Funcionalidades

- Crear tareas
- Marcar como completadas
- Eliminar tareas
- Modal para formulario

## Propósito

Proyecto de práctica para aprender React y manejo de estado.
